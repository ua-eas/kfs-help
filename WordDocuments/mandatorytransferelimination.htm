Mandatory Transfer Elimination

 >  >  >  >

The Mandatory Transfer Elimination document is used to determine whether an object code is considered a mandatory transfer code, an elimination code, or neither. This attribute of the object code is used for reporting purposes to easily identify object codes that need to be excluded or handled in a special way.
