Proposal Purpose 

 >  >  >  >

The Proposal Purpose document maintains categories that can be assigned to proposals to identify the type of work a project represents, such as 'Instruction' or 'Research. A previously non-existing Purpose can be added to the table or information for an existing one can be updated. The Purpose is primarily used with external sponsoring agencies for Contracts and Grants activities; however, it can also be used for awards internal to the institution if necessary.
