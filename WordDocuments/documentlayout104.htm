Document Layout

The Appointment Funding Reason Code document includes the Edit BC Appointment Funding Reason Code tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit BC Appointment Funding Reason Code tab definition

Title

 

Description

Appointment Funding Reason Code

The reason code for a requested salary appointment funding.

Appointment Funding Reason Description

Required. The description of the appointment funding reason code.

Active Indicator

 

Indicates whether this appointment funding reason code is active or inactive. Remove the check mark to deactivate this code.
