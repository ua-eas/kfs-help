Travel, Entertainment and Moving

Accessing Travel, Entertainment and Moving Functions

The Travel, Entertainment and Moving module allows users to create, view, and maintain a variety of documents. The KFS menus provide access to these documents as follows.

•      On the main menu:

o  The Transactions submenu provides access to standard documents for working with travel authorizations and reimbursements, entertainment expenses, and moving and relocation expenses.

o  The Administrative Transactions submenu provides access to an error correction document for agency transactions that have failed validation checks during import processing.

o  The Custom Document Searches submenu allows you to find and view existing reimbursement requests and initiate payments for them.

o  The Lookup and Maintenance submenu provides access to the TEM Profile document.

•      The maintenance menu offers access to a number of supporting documents. These documents are accessible via the Travel submenu.

•      The administration menu provides access to several upload functions for imported data.
