General Ledger > General Ledger Batch Processes > detectMissingEntriesJob

Parameters

The following parameters are used by the detectMissingEntriesJob.

 

detectMissingEntriesJob Parameters

Name

Description

ENTRY_GENERATING_DOCUMENT_TYPES

Specifies the Kuali Financials Document types that should generate general ledger entries.

LOOK_BACK_DAYS

Identifies the number of days the job will look back to determine if there are any missing entries.

NOTIFICATION_EMAIL_ADDRESS

The e-mail address to send notifications to when documents with missing ledger entries are found. If left blank, the job will log the documents.
