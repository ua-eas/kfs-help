Endowment Transaction Code Tab

The Endowment Transaction Code tab identifies specific Etran codes (if any) to be used as selection criteria when calculating fees.

     This tab is available for your use only if the Fee Type value is 'T' (transactions) in the Fee Method tab.

     If the Select by Endowment Transaction Code box is checked in the Fee Method tab, the Endowment Transaction Code tab must contain at least one Etran code for which the Include box is checked.

Endowment Transaction Code tab definition

Title

Description

Endowment Transaction Code

Required if Etran code is to be used as a criterion when calculating fees; otherwise, not used. Specifies an Etran code to be used.

Endowment Transaction Code Description

The description of, or name for, this endowment transaction code.

Include

Indicates whether this Etran code is to be used as a basis for fee calculation. Uncheck the box to discontinue using this Etran code for the fee calculations for this fee method.
