Edit Option

When you select the File Upload/Manual Edit Correction method, the system displays the edit button below the loaded data.

Figure 553 - Edit option in the Search Results tab

Clicking edit changes the tab to handle a manual edit.

Click edit corresponding to the row you want to edit (see below). Then make changes in the manual edit section with editable boxes below the loaded rows.

 

Figure 554 - The loaded rows are available for editing.

Figure 555 - Manual Editing section with data

Click edit when you finish editing the row. The changes you made are immediately reflected in the loaded row.
