Figure 220 - Proposal Purpose document

The Proposal Purpose document includes the Edit Proposal Purpose tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Table 11   Edit Proposal Purpose tab definition

Title

 

Description

Proposal Purpose Code

A one-letter code to represent the purpose. Examples are A=Instruction, C=Research, etc.

Proposal Purpose Description

Required. A text description of the proposal purpose in order to assist in determining which purpose to assign to the proposal.

Active Indicator

 

Indicates whether this proposal purpose code is active or inactive. Remove the check mark to deactivate this code.
