Collection Activity Report

 > > > >

The Collection Activity Report provides a means for AR Collectors to search for or print collection activity details.  From the on-screen search results, certain data in the collection activity event may be updated and/or new collection activities may be created.

       Displays when Contracts & Grants Billing is turned on.

       Only users with the AR Collector role can access the Collection Activity Report.
