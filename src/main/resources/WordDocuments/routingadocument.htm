Routing a Document

The document process supports both pre-established workflow routing and ad hoc routing. In workflow routing, Kuali Financials routes the document to the proper users based on business rules established in Workflow. Ad hoc routing allows a user to route the document to one or more individual users and/or groups for approval, acknowledgement, or FYI.

Unless you want to add an ad hoc routing, select Submit button to route the document in the predefined routing hierarchy.
