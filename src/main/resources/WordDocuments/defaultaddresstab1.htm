Default Address Tab

 

After an address is added, you have the option to set this address as the default for a specific campus.

 

Default Address tab field definitions

Title

Description

Set as campus default for

Select the campus from the pull down for which this address will be the default.

Active Indicator

Required. Select the check box if the default address is active. Clear the check box if it is inactive.
