Customer number

Display-only. The ID number for the customer. The system displays this value if a customer number exists on the referenced document. A change of customer number is possible via lookup for credits processed referencing the PO or the vendor.

    The lookup is not provided if a payment request number was entered in the Credit Memo Initiation tab.
