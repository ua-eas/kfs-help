Permissions

•      The chart code and organization assigned to the user creating the document must match the chart code and organization of the assets being modified by this document.

•      Permissions on the Asset Location Global document are based on the Kuali Identity Management (KIM) role assigned to the user. Capital Asset Management has five roles.

o  KFS-CAM Processor is a departmental user with access to assets with-in the hierarchy of related organization.

o  KFS-CAM Manager is the person responsible for managing inventory for a campus. This role has update access for assets where the physical campus code on the organization matches the campus of the manager.

o  KFS-SYS Asset Processor can edit all assets.

o  KFS-SYS Asset Manager can edit all assets.

o  KFS-SYS Plant Fund Accountant can edit all assets.
