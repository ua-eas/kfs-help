Lookup Criteria

 

Letter of Credit Draw Lookup screen definition

Field

Description

Document Number

 

The unique document number for the Letter of Credit Review document.

Letter of Credit Fund Group

 

Select a value from this list to return only Letter of Credit draws for a particular Letter of Credit Fund Group.

Letter of Credit Fund

Select a value from this list to return only Letter of Credit draws for a particular Letter of Credit Fund.

LOC Review Create  Date From

To search for LOC Review documents created during a particular period of time, enter the first date in the desired range of dates. You may also use the calendar icon  to select the date.

COC Review Create Date To

To search for LOC Review documents created during a particular period of time, enter the last date in the desired range of dates. You may also use the calendar icon  to select the date.

Amount Available to Draw

Enter the exact amount available to draw to search for or use >, <, >=, or <= to search for an amount available to draw range.

Claim on Cash Balance

 

Enter the exact claim on cash balance amount to search for or use >, <, >=, or <= to search for a claim on cash balance range.

Amount To Draw

Enter the exact draw amount to search for or use >, <, >=, or <= to search for a draw amount range.

Funds Not Drawn

Enter the exact funds not drawn amount to search for or use >, <, >=, or <= to search for a funds not drawn amount range.

Report Type

Select which report format you wish to create.  Selecting Amounts Not Drawn limits results to only amounts not drawn, and ignores other search criteria that may otherwise return additional data.
