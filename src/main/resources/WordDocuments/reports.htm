On the Main Menu tab, the Reports submenu provides access to a number of system reports.

Description

Account Labor Transactions

Account Status

Summary balances by Account Number and Object Code.

Account Transactions

Details of the individual transactions shown in summary in the Account Status report.

Consolidated Account Status

Groups all accounts for a given chart, fund, sub fund, and responsibility center.

Consolidated Status

Summary budget, actual, encumbrance and budget balance totals at the consolidation level for each chart, fund, sub fund, and responsibility center.

Reversion and Carry Forward Summary

Summary of budget balance reversions and carry forwards that occur during the organization reversion process.

Trial Balance

Combines beginning fund/net equity balances with additions, deductions, and ending balance for each account.
