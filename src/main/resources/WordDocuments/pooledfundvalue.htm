Pooled Fund Value 

 >  >  >

The Pooled Fund Value e-doc allows you to store historical unit values and distribution amounts for pooled funds. Using these e-docs you may update unit values for these funds and initiate the automated generation of distributions of cash and/or gains and losses to holders of the funds. The system uses data from these e-docs to generate automated pooled fund transactions in the overnight processes.
