View Related Documents Tab

In four of the TEM transactional documents—Entertainment, Moving and Relocation, Travel Authorization, and Travel Reimbursement—the View Related Documents tab lists other documents that are related to the document you are viewing. You may also use this tab to establish a relationship between two previously unrelated documents.

   If the DV was not created via the  button, it and any PREQs will need to be manually related before they are reflected in the Expense Totals tab.

View Related Documents tab definition

Title

 

Description

Related Document Number

Required. The document number of the document you are relating to the document you are viewing.

Description

Required. A description of the document or relationship.

To establish a relationship between two documents:

1.    Documents must be Saved in order to be linked.

2.    Click  at the top of the View Related Documents tab.

3.    Enter the document number that you want to link to this document into the Related Document Number field. It must be a valid document number.

4.    Enter a description of the relationship if desired.

5.    Click . The system displays the related document below the data entry line and enters the relationship in the View Related Documents tab of the other document as well.

6.    To associate another document with this one, repeat all steps.

7.    To sever a relationship between the document you are viewing and a related document, click delete on the line for the related document.
