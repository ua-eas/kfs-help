Document Layout

The Agency document includes the Edit Agency, Customer, Contracts & Grants, Addresses and Collections Maintenance tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

    The Customer, Contracts & Grants, Addresses and Collections Maintenance tabs will only display if Contracts & Grants Billing is turned on.
