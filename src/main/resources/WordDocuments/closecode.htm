Close Code 

 >  >  >  >

The Close Code Maintenance document is used to manage the various reasons that a KEMID may be closed. This information is used for reporting and tracking purposes and is always associated directly with a KEMID record. Possible values might include ‘Not funded,’ Refunded to Donor,’ ‘Expended,’ and ‘Transferred to an outside organization.’
