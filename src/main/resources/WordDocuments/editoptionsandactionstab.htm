Edit Options and Actions Tab

The Edit Options and Actions tab gives you options for the type of correction file you want to generate.

Edit Options and Actions tab definition

Title

 

Description

Process In Batch

Optional. Select the check box if you want the system to process this file as part of the G/L batch processing. Clear the check box if you want the system to process this file. A GLCP report will still be generated for the file even if it is not processed.

Output only records which match criteria?

Optional. Select the check box if you want to generate a group that contains only the records that are changed by the search and modification criteria. Clear the check box if you want to generate a group containing the records that match the search and modification criteria as well as those that don't match.

Show Output Group

This button acts as a switch and lets the user either view the output group after the changes on the GLCP document have been applied or view the original input group prior to any changes. The results are displayed in the Search Results tab.
