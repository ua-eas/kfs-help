Payments Lookup Tab

 

The only active field on this tab is the Click here link.

 

Payments Lookup tab field definitions

Title

Description

To view the payment lookup for this Asset

Link (Click here) to the Asset Payments lookup icon for this asset.

 

Clicking this link causes the system to open the Asset Payment Lookup with the Asset Number field pre-populated with the asset number of the asset the user is working with. The results table lists all payments that have been processed.

 

         If more than ten payments exist for the asset, the user may view all payments via the Asset Payment Lookup. For more information on the Lookup, see Asset Payment Lookup.
