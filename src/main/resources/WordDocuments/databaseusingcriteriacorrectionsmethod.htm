Database/Using Criteria Corrections Method

 

1.  Select Database and Using Criteria and click Select in the Correction Process tab.

The available files for editing appear in the Origin Entry Group list in the Documents in System tab.

2.  Select the file in the Documents in System list and click Load Group.

3.  Enter search and modification criteria in the Edit Criteria tab. Click Add as you add each criterion.

4.  Select the type of correction file to be created in the Edit Options and Action tab.

5.  Click Submit to process your LLCP document.

6.  Review the Summary tab.
