Create Capital Assets Section

This section contains two fields that are used to ensure that asset amounts do not exceed the total of the accounting line amounts selected for processing.

Create Capital Assets section definition

Title

 

Description

System Control Amount

Display only. The total amount available for distribution. As lines are selected in the Accounting Lines for Capitalization tab, the system adds the amounts on these lines to the system control amount.

System Control Remainder Amount

Display only. The amount remaining for distribution. As amounts are distributed to assets, the system reduces the remainder amount.
