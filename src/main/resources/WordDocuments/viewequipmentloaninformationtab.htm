View Equipment Loan Information Tab

This tab contains sections for information about the borrower and for the location at which the asset will be stored if different from the borrower’s address.

Figure 53 - View Equipment Loan Information tab

Table 46   View Equipment Loan Information tab definition

Title

 

Description

Loan Date      

The date the equipment loan was made.

Expected Return Date      

The anticipated date on which the asset will be returned.

  Assets cannot be lent for more than two years, so this date must not be later than two years from the loan date.

Return Date      

The date a loaned asset was returned.

Table 47   Borrower’s Address section definition

Title

Description

Name  

The borrower’s contact name.

Street      

The borrower's street address.

City      

The borrower's city.

State Abbreviation      

The borrower's state.

Postal Code      

The borrower's postal code.

Country Name      

The borrower's country.

Phone      

The ten-digit contact phone number for the borrower.

Table 48   Stored at Address section definition

Title

Description

Street      

The street address where the loaned asset is stored, if different from the borrower's address.

City      

The city where the loaned asset is stored, if different from the borrower's city.

State Abbreviation      

The two-letter code of the state where the loaned asset is stored, if different from the borrower's state.

Postal Code      

The postal code where the loaned asset is stored, if different from the borrower's postal code.

Country Name      

The country where the loaned asset is stored, if different from the borrower's country.

Phone
