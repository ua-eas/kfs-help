Financial Processing > Custom Document Searches

Financial Transactions

When you select Financial Transactions from the Main Menu, the system displays the Document Lookup screen. Although you may use this screen to search for any type of document, it initially displays fields that are especially helpful in searching for financial transaction documents.

Financial Transactions Document Lookup screen

Title

 

Description

Type

To search for all types of financial transaction documents, use the default entry of KFST. Or, to search for a particular type of financial transaction (Advance Deposit, Auxiliary Voucher, etc.) or any other type of document, enter the appropriate type code or use the lookup  to find it.

    If you change the Type entry and move the cursor to another field, the system displays fields appropriate to the type of document you have specified.

Initiator

 

To search for financial transaction documents initiated by a particular user, enter the person's username (principal name) or use the lookup  to find it.

Document/Notification ID

To search for a financial transaction document based on its unique document number, enter the number.

Date Created From

To search for financial transaction documents created during a particular period of time, enter the first date in the desired range of dates. You may also use the calendar icon  to select the date. The default is today's date.

Date Created To

To search for financial transaction documents created during a particular period of time, enter the last date in the desired range of dates. You may also use the calendar icon  to select the date.

Document Description

To search for financial transaction documents containing a particular description, enter the description.

Organization Document Number

To search for financial transaction documents containing a particular organization document number, enter the number.

    This number is typically used to identify a related record in another campus system (for example, a customer relationship management system, an order processing system, etc.).  When used in this way, the entry ties a KFS cash control document to a record in the non-Kuali application.

Chart Code

 

To search for financial transaction documents associated with a particular chart, select the appropriate chart from the list or use the lookup  to find it.

Account Number

To search for financial transaction documents associated with a particular account, enter the appropriate account number or use the lookup  to find it.

Organization Code

 

To search for financial transaction documents associated with a particular organization, enter the appropriate organization code or use the lookup  to find it.

Total Amount

To search for financial transaction documents based on the total amount, enter the amount.

Search Result Type

To view financial transaction documents in their entirety, click Document Specific Data. To retrieve only the routing status, date created, and route log for these financial transaction documents, click Workflow Data.

Name this search (optional)

To save this set of search criteria for later use, enter a name for it.

 

   In many fields you may use special characters to search on a partial entry. For information about using special characters to search, see Data Entry Tips.

   For information about using features and options that are common to all types of custom document searches and working with the search results, see Performing Custom Document Searches.

After you select a financial transaction document from the search results, the system displays it in view-only mode.

   For information about the fields on a financial transaction document, see “Document Layout” in the section on the appropriate document type.
