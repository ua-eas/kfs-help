Global Organization Reversion Details Tab

 

The Global Organization Reversion Details tab works in the same manner as the Organization Reversion Details tab of the Organization Reversion document. It contains object codes and carry forward/reversion rules by organization reversion category. Fields indicated as required are required only if that organization reversion category is completed.

 

This tab must be completed regardless of the Carry Forward Indicator in the Organization Reversion tab.
