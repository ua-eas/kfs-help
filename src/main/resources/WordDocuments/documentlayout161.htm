Document Layout

The Report Period Status Code document includes the Edit Report Period Status Code tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Report Period Status Code tab definition

Title

 

Description

Report Period Status Code

A single letter code uniquely identifying this report period status.

Report Period Status Description

Required. A text description of the status code. Default values are:

N = Not yet opened (updates allowed)
O = Open (batch process has generated effort reports)
C = Closed (no updates allowed)
