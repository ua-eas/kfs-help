KEM_ECI_ECDD_ECT_Import.xls

The KEM_ECI_ECDD_ECT_Import.xlsx template applies to the Cash Increase (ECI), Cash Decrease (ECDD), and Cash Transfer (ECT) document types. For each of these e-docs, this template is used to import transaction lines.

The basic format of the template is shown below.
