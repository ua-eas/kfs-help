Registration Code 

 >  >  >

The Registration Code document indicates where a security is held (e.g., custodial bank). This code is used on holding tax lot records and is used to provide an accurate analysis of KEMID holdings in history. The registration code associated with each current holding record is input as a part of the security transaction that creates or updates the holding record.
