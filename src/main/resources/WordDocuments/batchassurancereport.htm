General Ledger

Batch Assurance Reports

 

As discussed in the previous sections, a number of text reports are generated by each step of the batch programs to give you assurance that the transactions are processed in order. Each of the generated text reports is suffixed by the date and time of run. The following table summarizes the batch assurance reports generated by the batch process:

 

Batch assurance reports

Report Title

Text Report Name

Description

ERROR AND STATISTICS REPORT

balancing

Automated balance reconciliation. The process builds a series of entry and balance tables that are synched with your General Ledger as they are created. Thereafter,the process uses these history tables, adding transactions from each nightly batch cycle. Created by the posterBalancingJob.

Collector Reports

collector

Summary of all files processed by the collector, a list of all errors from collector transactions, statistics for records read and in error, and an input summary of all valid transactions by balance type, origin code, fiscal year and period code. Created by the CollectorJob.

Enterprise Feed Report

enterprise_feed

Summarizes files successfully loaded and files that were not sucessfully loaded. Created by the enterpriseFeedJob.

GLPE Statistics Report

glpe_ledger

Summary total and record count of approved General Ledger pending entries fed by balance type, origin code, fiscal year and period. Created by the nightlyOutJob.

PENDING ENTRY LEDGER TABLE

glpe_list

Detailed listing of approved General Ledger pending entries fed. Created by the nightlyOutJob.

GL POSTER REPORT (MAIN)

posterJob

Total record count of rows deleted, inserted, and updated from in the GL tables updated by the poster. Created by the PosterJob.

Main Poster Error Transaction Listing

posterJob

Detailed listing of the error transactions along with a summary total and record countfor transactions not posted. Created by the PosterJob.

Main Poster Input Transactions

posterJob

Summary total and record count of transactions posted fed by balance type, origin code, fiscal year and period. Created by the PosterJob.

GL POSTER REPORT (REVERSAL)

posterJob

Summary total and record count of rows selected, deleted, inserted, and updated in the GL tables by the reversal poster. Created by the PosterJob.

Reversal Poster Transaction Listing

posterJob

Detailed list of transactions generated by the reversal posteralong with a summary total and record count. Created by the PosterJob.

Reversal Poster Error Transaction Listing

posterJob

Detailed listing of error reversal transactions along with a summary and record count for transactions not posted. Created by the PosterJob.

Reversal Poster Input Transactions

posterJob

Summary total and record count of transactions that generated reversals by balance type, origin code, fiscal year and period. Created by the PosterJob.

ICR GENERATION REPORT

posterJob

Summary total and record count of rows selected, deleted, and kept from the GL Expenditure Transaction table and the record count of ICR transactions generated. Created by the PosterJob.

GL POSTER REPORT (ICR)

posterJob

Total record count of rows deleted, inserted, and updated in the GL tables by the ICR poster. Created by the PosterJob.

ICR Poster Error Transaction Listing

posterJob

Detailed listing of ICR error transactions along with a summary and record count for transactions not posted. Created by the PosterJob.

ICR Poster Input Transactions

posterJob

Summary total and record count of ICR transactions generated by the ICR poster by balance type, origin code, fiscal year and period. Created by the PosterJob.

GL Budget Balance Summary

posterJob

Beginning, annual, monthly and accumulated balances by fund group for budget balance types CB, BB, MB. Created by the posterJob.

GL Actual Balance Summary

posterJob

Beginning, annual, monthly and accumulated balances by fund group for the AC balance type. Created by the posterJob.

GL Encumbrance Summary

posterJobb

Beginning Balance, Annual Balance, CB Beginning Balance and YTD Balance by fund group for the encumbrance balance types EX, IE, PE, CE. Created by the posterJob.

Poster Output Summary

posterJob

Summary debit, credit, budget, and net amount total by balance type, fiscal year, period code and fund group. Created by the posterJob.

GL POSTER REPORT (ICR Encumbrance)

posterJob

Total record count of rows deleted, inserted, and updated in the GL tables by the ICR Encumbrance poster. Created by the PosterJob.

ICR Encumbrance Poster Error Transaction Listing

posterJob

Detailed listing of ICR Encumbrance error transactions along with a summary and record count for transactions not posted. Created by the PosterJob.

ICR Encumbrance Poster Input Transactions

posterJob

Summary total and record count of ICR Encumbrance transactions generated by the ICR poster by balance type, origin code, fiscal year and period. Created by the PosterJob.

GL SCRUBBER REPORT

scrubberJob

Summary total number of records scrubbed and automated entries generated (offset, capitalization, liabilities, plant indebtedness, and cost share). Errors detected by the scrubber are also listed in detail. Created by the scrubberJob.

 

GL DEMERGER REPORT

scrubberJob

Total number of valid and invalid transactions and scrubber-generated entries (offset, capitalization, plant indebtedness, cost share) bypassed. Created by the scrubberJob.

Scrubber Input Transactions with Blank Balance Types

scrubberJob

Detailed listing of transactions with bad or blank balance types. The scrubber replaces these with a balance type of AC. Created by the scrubberJob.

Error Listing - Transactions Removed From the Scrubber

scrubberJob

Detailed listing of transactions removed from 'Scrubber' including transactions pulled by the 'Demerger' process. Also includes subtotals for error entries. Created by the scrubberJob.

Ledger Report

scrubberJob

Summarylisting of transactions scrubbed, including GLPE and external batch feed files by balance type, origin code, fiscal year and period. Created by the scrubberJob.

SUFFICIENT FUNDS REBUILDER REPORT

sufficient_funds

Detailed listing of sufficient funds errors (if any) and a summary of records read, deleted, converted or retained due to errors by the sufficient funds rebuilder process. Also records the number of entries added, deleted and updated in the sufficient funds table. Total and record count of Sufficient Balance Table rows selected, deleted, updated, and inserted. Created by the sufficientFundsFullRebuildJob.

 

       The delivery mechanism of these reports depends on the implementing institution.
