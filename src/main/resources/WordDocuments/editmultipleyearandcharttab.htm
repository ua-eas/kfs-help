Edit Multiple Year and Chart Tab

There are two ways to populate the object codes that you want to update globally in the Edit Multiple Year and Chart tab. One is by manually entering or selecting one object code at a time from the normal Object Code lookup , the other is by using a special multiple value lookup called the Look Up / Add Multiple Object Code Lines  to return multiple values.
