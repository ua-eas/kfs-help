Document Layout

The Class of Service document includes the Edit Class of Service tab.

Edit Class of Service tab definition

Title

Description

Class of Service Code

Required. A unique code identifying a class of service.

Expense Type Meta Category

Required. The Expense Type Meta Category associated with this class of service.

     Meta Categories are hard coded values that identify expenses that have special rules when used, such as when to display the Class of Service or Airfare Source codes or zeroing out breakfast in the Per Diem table if used in the Expenses tab. Adding new meta categories requires coding. The Meta Categories represent traditional travel expenses, such as Airfare, Rental Car, Breakfast, Lunch, Dinner, etc.

Class of Service Name

Required. The descriptive name assigned to this class of service.

Special Request Review Required

Check the box to indicate that TEM Documents that use this class of service must route to the Special Request Review role.

Active

Indicates whether this Class of Service record is active (checked) or inactive (unchecked).
