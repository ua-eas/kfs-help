Tickler Tab

This tab contains basic information for the tickler, such as the reason for it, the text of the message, and the frequency or dates for its transmission.

Tickler tab definition

Title

Description

Tickler Type Code

Required. Indicates the reason a person or department wants or needs to be notified of an upcoming event or action to be taken. These reasons are defined by your institution through Tickler Type Code maintenance e-docs. Different type codes might indicate ‘Investment Review’, ‘Spending Review’, or Income Reinvestment.’ Enter an existing code or use the lookup  to find it.

Tickler Frequency

Indicates how often you want the tickler to be sent. Enter an existing frequency code or use the lookup  to find it. You must enter either a Tickler Frequency value or a Tickler Next Due Date value, but you may not enter both. If you do not enter a Tickler Frequency value, the system assumes that this is a one-time reminder so you must enter Tickler Next Due Date value.

Tickler Next Due Date

Indicates a specific date on which the next tickler is to be sent. Enter the date or select it from the calendar . You must enter either a Tickler Frequency value or a Tickler Next Due Date value, but you may not enter both. If you do not enter a Tickler Frequency value, the system assumes that this is a one-time reminder so you must enter Tickler Next Due Date value.

Tickler Detail

Required. The text of the tickler message.

Tickler Entry Date

Display only. When you create a tickler, the system automatically enters the current date in this field.

Tickler End Date

The date the tickler expires. Enter the date or select it from the calendar . If you mark a tickler as inactive, the system automatically populates the Tickler End Date based on the current date.

Active Indicator

 

Required. Indicates whether this tickler is active or not. Check the box to activate the tickler. Clear the box to deactivate it. If you mark a tickler as inactive, the system automatically populates the Tickler End Date based on the current date.
