ACH Bank

 

The ACH Bank document allows you to maintain a valid list of banks for ACH payees.

 

       The ACH Bank table can be updated automatically by using the pdpLoadFederalReserveBankDataJob batch job. This document allows you to manually add or edit entries if necessary.

       Only users with the proper permissions may create, edit or view bank account numbers.
