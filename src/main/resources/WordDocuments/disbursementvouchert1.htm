Disbursement Voucher Tax Income Class

 >  >  >  >

The Disbursement Voucher Tax Income Class document is used to establish the valid income class codes that appear on the Nonresident Alien Tax tab of the Disbursement Voucher document. This code reflects the type of activity the disbursement is for, applying a category to the services or goods for which funds are being disbursed. Examples might include fellowships or royalties.

To access the Disbursement Voucher Tax Income Class document, select Disbursement Voucher Tax Income Class from the Financial Processing group on the Maintenance menu tab.
