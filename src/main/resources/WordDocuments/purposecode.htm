Purpose Code 

 >  >  >

The Purpose Code Maintenance document allows you to manage codes used to categorize KEMIDs according to their allowable use of funds. Each purpose code may indicate a very specific limitation or a broad range of uses, depending on the donor’s wishes for a gift fund. So, for example, you might set up different purpose codes for scholarships, professorships, chairs, and/or general use.
