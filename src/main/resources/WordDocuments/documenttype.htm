Document Type

 

The Document Type document defines basic information about the document types that exist in Kuali Financials. It also defines document types specific to workflow and KIM. Many definitions of a document type not defined here are included in its workflow process definition and the data dictionary (a technical resource). The workflow process definition for a document type can be viewed using the Document Type lookup.
