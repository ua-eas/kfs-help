Estimated Per Diem Expenses Tab

 

This tab allows you to calculate allowable expenses for meals, incidentals, lodging, and meals when per diems or set rates are used for reimbursement of trip expenses.

 

       The Per Diem Expenses tab is also used by the Travel Reimbursement transaction document. For detailed information about it, see Common Tabs in Transaction Documents: Per Diem Expenses Tab.
