Note that multiple Campus Code/Contract Manager combinations may be assigned to the commodity code by clicking add after entering data for each combination.

Contract Managers field definitions

Title

Description

Active Indicator

Indicates whether this commodity code is active or inactive. Remove the check mark to deactivate.

Campus Code

Required. The campus for the commodity code. Existing campus codes may be retrieved from the list or from the lookup icon.

Contract Manager

Required. The contract manager for the commodity code. Existing contract managers may be retrieved from the list or from the lookup icon.
