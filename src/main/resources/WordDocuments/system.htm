System

This section provides information about using the Kuali Financial System administration functions available via the user interface and the documents, reports, and other functions that pertain to more than one module in Kuali Financials.

 

The Kuali Financial System enables users with proper authorization to perform powerful functions that affect the entire system. These functions allow you to:

•      Support the Kuali Identity Management (KIM) module and implement access security based on accounting line attributes

•      Define basic types of location and reference information

•      Control the running of batch jobs and

•      Configure Kuali Financials.

 

      Several Rice and Workflow related options are available from the Kuali Financials user interface, however, they are intended to support technical activities and are documented in the Technical User documentation and will not be covered in this guide.

      In order to work efficiently, you need to understand the basics of Kuali Financials user interface. For information and instructions about logging on and off, navigating, understanding the components of screens, and performing basic operations in the screens, see Overview

       As you work in the system management screens covered in this guide, keep in mind that information presented here is also available via Kuali Financials online help.
