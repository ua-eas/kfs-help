Database/Remove from Processing

 

This option pulls the file from the labor distribution cycle, excluding it from future processing.

1.  Select Database and Remove Group From Processing and click Select in the Correction Process tab.

The available files for editing are displayed in the Origin Entry Group list in the Documents in System tab.

2.  Highlight the Origin file that you want to remove from processing and click Remove Group From Processing.
