Routing

 

•      If the document is initiated by the traveler, the document does not route.

•      TEM Profiles route to Central Profile Reviewer for approval. If the traveler is an employee, they will receive an FYI notification.

•      If the NRA box is checked, the document routes to the Tax Manager role.

•      If either the NRA box or the Citizenship entry is changed, the document routes to the Tax Manager role.
