View Asset

 

The view action is available from the search results table for an asset lookup for retired assets. Choosing this action causes the system to display a read-only version of the Asset Inquiry document. This document presents all information that is stored in CAM about the asset. You may also view the asset record by clicking the underlined asset number for it in the search results list.

 

          The view operation allows you to see all information about the asset but does not allow you to change field entries.
