Removing Menu Items

 

Only Menu Items that were added via Navigation Configuration can be deleted. If you wish to remove existing menu items, remove the initiate document permission for this item and it will no longer display in the left hand navigation. It will continue to show in the Navigation Configuration. You may want to create a Not Used group to store these links.

 

To remove items manually added:

•      Click the Group Label that contains the menu item you want to delete. The list of menu items will open to the right of the navigation.

•      Click the trashcan icon to remove the item from the left hand navigation.
