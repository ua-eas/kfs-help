Example

The University of Kuali has a group of employees that can potentially take a leave of absence without pay for a period of nine months. The university wants to track this type of leave in the base budget. A maintenance user for the University uses a Budget Construction Appointment Funding Duration maintenance document to create a new duration type, entering Appointment Duration Code 'LWP9' and Appointment Duration description 'LWOP: 9 months.'
