Edit Security Principal tab

This tab allows the user to identify a principal to which the security model(s) and security definition(s) apply.

The display above shows the tab in create mode. In edit mode, this tab presents a display-only set of fields on the left and editable fields on the right in which the user may enter changes.

Edit Security Principal tab definition

Title

Description

Principal Id

Display only. The unique identifying code assigned to a principal.

Principal Name

Required. The username of an individual with which one or more security models and/or definitions is associated. Enter a value or select it from the lookup .

    Clicking the lookup  causes the system to display the KIM Person Inquiry screen.
