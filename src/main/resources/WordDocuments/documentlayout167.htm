Registration Code Maintenance document

The Registration Code Maintenance document includes the Edit Registration Code tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Registration Code tab definition

Title

Description

Registration Code

Required when initiating a document; otherwise, display only. Indicates where a security is held in custody for the institution. The code may reference a bank, a broker or a location in your institution.

Registration Code Description

Required. The description of the custodial location.

Active Indicator

 

Indicates whether this registration code is active or inactive. Remove the check mark to deactivate the code.
