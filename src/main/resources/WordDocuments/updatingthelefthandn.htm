Updating the left hand navigation

 

When you edit or add a new Menu Item, the following fields are available.

 

Left Hand Navigation Menu Items field definitions

Title

Description

Link Name

Required: Enter the name of the new Menu Item. This will display in the left hand navigation of the dashboard.

URL

Required: Enter the URL of the Menu Item.

    Only absolute path names are allowed.

Category

Select the appropriate Navigation Category.

Group:

Select the group that this item should be a part of.
