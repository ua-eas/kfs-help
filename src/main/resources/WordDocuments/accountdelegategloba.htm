Account Delegate Global

      For general information about accessing and working with global documents, see Global Chart of Accounts Documents.

The Account Delegate Global document allows you to create delegates for multiple document types on one or more accounts on a single document. Unlike the other global documents, you may initiate it by entering information from scratch or by populating the document from a model you choose. Although the use of a model document is not a requirement, it is a convenient way of initiating the Account Delegate Global document when you have many delegates to define.
