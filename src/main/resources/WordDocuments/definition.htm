Definition

 >  >  >  >  >

The Security Definition document is used to define security attribute restrictions and the types of documents to which they apply. Once activated, a definition may be assigned to a model or principal.

When the user chooses the Definition option from the menu, the system displays the Security Definition Lookup screen. After the user searches for and selects a definition to edit or clicks the create new button, the system presents the Security Definition document.
