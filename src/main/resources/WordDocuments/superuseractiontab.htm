Super User Action Tab

Super user actions let you move a document past workflow nodes where it may be held up because another user is unavailable to take action on it or due to a system problem. Super user actions are an administrative tool and safety net. Super user action can be taken via the super user search or the Super User Action tab on the document. Access to the Super User Action tab is granted via the Super User Approve Document permission.

     For more information about the Super user Search, see Super user vs. Non-Super user Searches and Super user Functions.
