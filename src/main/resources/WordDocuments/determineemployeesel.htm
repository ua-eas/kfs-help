Determine Employees Eligible for this Report

 

Employees with Labor Ledger entries of Balance Type AC or A2 within the payroll fiscal year and period specified for this report will be included if those entries correspond to an earn code and pay group combination associated with this report's report type (as defined in the Report Earn Paygroup table.)

 

This identifies the initial subset of employees who might be eligible to have Labor Ledger data extracted. This set will be further pared down in the steps that follow.
