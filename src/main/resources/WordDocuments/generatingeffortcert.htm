Effort Certification

Generating Effort Certification documents

 

Effort certification documents are created via batch with the following jobs:

1.    Extracting data from the Labor Ledger (the EffortCertificationExtractJob).

2.    Creating effort reports from this extracted data (the EffortCertificationCreateJob).
