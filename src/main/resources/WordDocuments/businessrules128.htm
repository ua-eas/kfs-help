Chart of Accounts > Chart of Accounts Documents > Sub-Account > Process Overview
Business Rules

•      Only authorized users can modify the sub-account type code.

•      If sub-account type code is 'CS,' then the fields Cost Sharing Chart of Accounts Code and Cost Sharing Account Number in the CG Cost Sharing tab are required.

•      If sub-account type code is 'EX' and the Account associated with the Sub-Account is a Contracts and Grants account, then all fields in the CG ICR tab are required.

•      If any field in the Financial Reporting Code tab is completed, all fields become required.
