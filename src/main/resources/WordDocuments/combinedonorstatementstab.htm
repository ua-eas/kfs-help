Combine Donor Statements Tab

Combine Donor Statements tab

This tab specifies any KEMIDs whose statements are to be combined with statements for the current KEMID. Use of this tab is not required for any KEMID record. If you choose to use the tab, you may add one or more Combine Donor Statements records to a KEMID.

Combine Donor Statements tab definition

Title

Description

Combine Donor Sequence Number

Display only. The system-assigned sequence number for this record.

Combine with KEMID

Required. The KEMID whose statements are to be combined with statements for this KEMID. Enter an existing KEMID or use the lookup  to find it.

Combine with KEMID Short Title

Display only. The common name of the specified KEMID.

Combine Date

The first date on which statements are to be combined. Defaults to the current date but you may enter another date or select it from the calendar .

Terminate Combine Date

The last date on which statements are to be combined. Enter the date or select it from the calendar . After this date is reached, this record is no longer available for viewing or editing.
