Document Layout

The Purpose Code Maintenance document includes the Edit Purpose Code tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Purpose Code tab definition

Title

Description

Purpose Code

Required when initiating a document; otherwise, display only. A code representing the allowable use of funds for a KEMID.

Purpose Code Description      

Required. Your institution’s description of this purpose code.

Active Indicator

 

Indicates whether this purpose code is active or inactive. Remove the check mark to deactivate this Purpose Code record.
