Customer XML Upload Format

 

      For information on procedures and formatting rules that apply to all batch uploads, see Batch Upload Basics .

 

Keep these rules in mind:

•      The customer upload file must begin with the standard XML version line such as:

•      The root (first) tag for Customer files must be the customers tag, and the file can contain only one.

•      The customer tag follows the customers tag and contains other identifier tags that describe an AR customer.

•      The customer tag can be used multiple times.

•      Each customer tag must have at least one address tag and multiple addresses can be defined.
