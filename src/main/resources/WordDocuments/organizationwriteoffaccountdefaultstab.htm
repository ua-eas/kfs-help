Organization Writeoff Account Defaults Tab

 

Organization Writeoff Account Defaults tab field definitions

Title

Description

Account Number

The write-off account number default for this organization. Existing account numbers may be retrieved from the lookup.

Chart Code

The Chart of Accounts code for the writeoff account default for this billing organization. Existing chart codes may be retrieved from the lookup.

Object Code

The writeoff object code default for this organization. Existing object codes may be retrieved from the lookup.

Organization Reference Identifier

Optional. The organization reference identifier if applicable.

Project Code

The write-off project code default for this organization. Existing project codes may be retrieved from the lookup.

Sub Account Number

The writeoff sub account number default for this organization. Existing sub account numbers may be retrieved from the lookup.

Sub-Object Code

The write-off sub object code default for this organization. Existing sub object codes may be retrieved from the lookup.
