Message of the Day

All three menu tabs include the Message of the Day at the top of the screen to quickly distribute messages to all KFS users on-line.

     Updating the Message of the Day: If no message of the day is defined in the system, the Message of the Day area is not displayed on the screen. To define the message of the day, initiate the Message of the Day e-doc from the Administration menu tab. See Message of the Day in the Guide to the KFS Administration Menu.
