Message of the Day

The Message of the Day appears at the top dashboard to quickly distribute messages to all Kuali Financials users.

       Updating the Message of the Day: If no message of the day is defined in the system, the Message of the Day area is not displayed on the screen. To define the message of the day, initiate the Message of the Day document.
