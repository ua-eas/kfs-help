Data Mapping Field Definition

 >  >  >  >

The Data Mapping Field Definition lookup displays definitions related to a given field in KFS. This information includes definitions such as the functional field description, the length of the field, whether or not it is encrypted, and the name of the field at the database level

     This data can only be updated by a technical resource and many of the values here are only meaningful to a technical user.
