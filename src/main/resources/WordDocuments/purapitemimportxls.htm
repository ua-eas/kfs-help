PURAP_Item_Import.xls

The PURAP_Item_Import.xls template applies to the Requisition document type and is used to import items into the requisition.

The basic format of the template is shown below.

PURAP_Item_Import template format

Column

Field Name

A

Quantity

B

UOM*

C

Catalog Number

D

Commodity Code

    If your institution has set the ENABLE_COMMODITY_CODE_IND or  ITEMS_REQUIRE_COMMODITY_CODE_IND parameter to NO, delete the Commodity Code column before using this template.

E

Description*

F

Cost*
