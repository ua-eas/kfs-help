Subclass Code 

 >  >  >

The Subclass Code Maintenance document allows you to further identify the type of security that is represented by a record. Subclass codes enable your institution to establish codes in a different scheme for external reporting purposes with as much detail as needed. For example, a class code might represent real property and one of its subclass codes might represent mineral rights.
