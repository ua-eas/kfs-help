Contracts & Grants > Attribute Maintenance Documents > Research Risk Type

Document Layout

The Research Risk Type document includes the Edit Research Risk Type tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Risk Type Code tab definition

Title

Description

Research Risk Type Code

A code to uniquely identify the research risk type.

Research Risk Type Description

A description for the research risk type code.

Control Attribute Type Code

A control attributes type code to be associated with this research risk type.

Sort Number

Required. A number to control the list position where this control attribute appears.

Research Risk Type Notification Value

A value to route to the notification group. Valid options are 'Y' (yes), 'N' (no), 'A' (all), 'X' (never).

Active Indicator

 

Indicates whether this research risk type code is active or inactive. Remove the check mark to deactivate this code.
