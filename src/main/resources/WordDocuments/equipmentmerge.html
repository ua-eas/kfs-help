Equipment Merge

 

Occasionally, physical inspection reveals that two or more assets have been created that should be combined (merged) into one asset because they work as a system. Using the merge function, users may combine these assets.

 

To merge two or more assets, the user chooses the merge link for the target asset—that is, the surviving asset—from the Actions column of the Asset Lookup results table. The system displays the Asset Retirement Global document.
