Sensitive Data

The system will check the Notes field, Document Overview Explanation and Routing Annotations to determine if sensitive data has potentially been added by the user when creating a document.

The delivered configuration restricts the use of Social Security Numbers and Credit Card Numbers defined in parameter SENSITIVE_DATA_PATTERNS.

If parameter SENSITIVE_DATA_PATTERNS_WARNING_IND is set to N, the user will be presented with an error and will not be able to continue until the data is removed or corrected. If set to Y, a warning is presented and the user can continue.
