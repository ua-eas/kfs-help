Type Code 

 >  >  >

The Type Code Maintenance document allows you to manage codes used to organize KEMIDS into groups for reporting purposes. For example, your institution's groups might include University General Endowed Funds, Discretionary Funds and Expendable Funds. Data stored in each Type Code record also provides default information for fields in the KEMID records to which the type code is assigned. Finally, through these records, users may reset fees to be automatically assessed against KEMIDs.
