Asset Depreciation Convention

 >  >  >  >

The Asset Depreciation Convention document associates a Financial Object Sub-Type Code with one of two depreciation convention codes (HY- Half Year Convention and FY - Full Year Convention). The system uses this information to calculate the date depreciation begins for a capital asset.
