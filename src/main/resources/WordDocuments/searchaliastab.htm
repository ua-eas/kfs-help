Search Alias Tab

 

The Search Alias tab is used to define other names that may be used when searching for this vendor.

 

Search Alias tab field definitions

Title

Description

Active Indicator

Required. Select the check box if the search alias name is active. Clear the check box if it is inactive.

Search Alias Name

Required. Enter the alternate name that may be use to search for this vendor.
