Suspense Account Logic

•      If parameter SUSPENSE_ACCOUNT_LOGIC_IND is set to Y, the Labor Ledger Scrubber will attempt to post to the accounting string specified in the SUSPENSE parameters when no other valid account can be found. If N, and no other valid account can be found, the Labor Ledger Scrubber will reject the transaction.
