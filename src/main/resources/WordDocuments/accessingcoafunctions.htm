Accessing COA Functions

Users maintain the KFS Chart of Accounts tables via a number of different documents. Some of these documents are accessible from the Main Menu while others are accessible from the Maintenance Menu.

•      Standard COA documents (defining accounts, object codes, etc.) are accessed via the Lookup and Maintenance submenu on the Main Menu tab.

     For more information about these documents, see Standard COA Documents.

•      Supporting COA documents (for account types, balance types, etc.) are accessed via the Chart of Accounts submenu on the Maintenance menu tab. (All KFS users may look up valid values in these documents, but only members of the KFS-SYS Manager role may copy, edit or create new documents.)

     For more information about these documents, see COA Attribute Maintenance Documents.
