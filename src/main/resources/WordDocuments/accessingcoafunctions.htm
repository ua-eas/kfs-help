Accessing COA Functions

Users maintain the KFS Chart of Accounts tables via a number of different e-docs. Some of these e-docs are accessible from the Main Menu while others are accessible from the Maintenance Menu.

•      Standard COA e-docs (defining accounts, object codes, etc.) are accessed via the Lookup and Maintenance submenu on the Main Menu tab.

     For more information about these e-docs, see Standard COA E-Docs.

•      Supporting COA e-docs (for account types, balance types, etc.) are accessed via the Chart of Accounts submenu on the Maintenance menu tab. (All KFS users may look up valid values in these e-docs, but only members of the KFS-SYS Manager role may copy, edit or create new e-docs.)

     For more information about these e-docs, see COA Attribute Maintenance E-Docs.
