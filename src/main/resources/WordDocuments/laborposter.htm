Labor Poster

 

The Labor Poster writes valid entries to the Labor Ledger Entry table, updates balances in the Labor Ledger Balance table and summarizes the entries for posting to the Labor Ledger.

 

Several reports are generated by the labor poster process and are combined into a single report (laborPosterJob)

 

•      Labor Poster Error Listing

•      Labor Poster Summary Report

•      Labor Poster Report

•      Labor Poster Ledger Report ‘

•      Labor Poster Output Summary

•      Labor Actual Balance Summary

•      Labor Budget Balance Summary

•      Labor Encumbrance Summary
