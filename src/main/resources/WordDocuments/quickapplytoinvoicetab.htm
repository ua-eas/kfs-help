Quick Apply to Invoice Tab

This tab allows you to quickly select multiple invoices and apply funds sufficient to pay each selected invoice in full.

Quick Apply to Invoice tab definition

Title

 

Description

Invoice Number

The number of an open customer invoice.

Open Amount

The open amount on the invoice.

Quick Apply

Check the box for each invoice you will fully pay with funds on this Payment Application document.
