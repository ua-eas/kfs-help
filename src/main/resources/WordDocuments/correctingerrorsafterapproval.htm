Correcting Errors after Approval

The error correction action allows you to correct documents by creating a new document that reverses the original transaction that has been fully approved. A document created with the error correction action must route and be approved in the same manner as the document it corrects.

   The error correction action should not be confused with the financial transaction document type General Error Correction (GEC), which is described in General Error Correction .

1.  Click Error Correction button.

2.  The system creates a new document with a new document ID. The system also displays Corrects Document ID in both the document header and the Notes and Attachment tab of the document.

Amounts are in negative to reverse the original transaction.

The new document has an annotation that is an error correction.

3.  Click Submit button.

The header of the corrected document shows the corrected by document ID.
