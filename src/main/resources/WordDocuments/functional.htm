Administration Menu > Configuration

Functional

>>

The Functional configuration submenu group provides access to high-level configuration options available to functional users. These include functions that allow authorized users to maintain the message of the day and various parameters and business rules, define many high-level attributes of a fiscal year, and perform other functions that affect users and transactions throughout the system. Some functions are only lookups, such as the ability to view data mapping fields and others are inquire-only screens like the Routing & Identity Management Document Type Hierarchy.

Table 52  Functional submenu group

Document

 

Description

Data Mapping Field Definition

Document Type

Functional Field Description

Provides a functional description for a given field in the application.

Message of the Day

Maintains the message displayed on the screen.

Parameter

Defines parameters and business rules in the KFS.

Parameter Component

Defines valid components of the system to which parameters can be related.

Parameter Type

Defines the basic types of parameters that can be created.

Routing & Identity Management Document Type Hierarchy

This special view combines KIM data with the KFS Document Type Hierarchy to display responsibility and permission information by document type.

System Options

Defines many high-level attributes of a fiscal year, including balance types, object types and the university chart level.
