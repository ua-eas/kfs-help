Summary Tab

The system automatically generates entries in the Summary tab after the LLCP document is submitted. This tab contains summary information about the debits and credits affected and the number of rows of data that the LLCP document has output. It also identifies the file name of the original input file and the newly created output file (after the document reaches 'FINAL' status).

The output entries generated by the LLCP document will also be displayed.
