W-8 Ownership Types tab definition

Title

Description

Ownership Code

Required. The Ownership Code that identifies the Ownership Type that should be allowed for this W-8 Type. Enter a valid value or use the  icon to return a value.

Active Indicator

 

Indicates whether this Owership Type is active or inactive. Remove the check mark to deactivate.
