Transaction Statement 

  >   >>  >

The Transaction Statement Generation screen allows you to generate transaction statements that reflect cash-based activity for a KEMID or group of KEMIDs from the beginning of a month through the end of the same or another month.

     This screen generates the detail view for each individual KEMID.  It does not provide a total view for all KEMIDS in a group.
