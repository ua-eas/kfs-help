File Upload/Using Criteria Corrections

 

1.  Select File Upload and Using Criteria; then click Select in the Correction Process tab.

2.  Browse for and load the file in the Correction File Upload tab.

3.  Enter search and modification criteria in the Edit Criteria tab. Click add as you add each criterion.

4.  Select the type of correction file to be created in the Edit Option and Action tab.

5.  Click Submit to process the LLCP document.

6.  Review the Summary tab.
