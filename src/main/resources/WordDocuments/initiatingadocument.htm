Initiating a Document

1.     Select the document link from the main menu.

If you have not logged into the KFS, the system prompts to enter your user ID and password.

2.     Complete required tabs for the document.

    For information about the requirements for a particular type of e-doc, see the help documentation for the specific document type.

3.     Click  to route the document for approval.
