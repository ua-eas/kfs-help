Business Rules

The following business rules apply to the Credit Card or Agency document:

•      Payment indicator can only be checked when Travel Card Type Code represents a Corporate Card.

•      When Payment indicator is checked, Vendor Number and Bank Code are required and payment will be extracted for processing by Pre-Disbursement Processor after the TEM document to which the imported expense was reconciled is FINAL.
