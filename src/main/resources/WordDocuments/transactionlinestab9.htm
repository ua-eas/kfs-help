Transaction Lines Tab

Transaction Lines tab

This tab contains one or more lines that specify the KEMID(s) for both the From and To sides of the transaction. It also includes all relevant financial details for the transaction as it applies to each KEMID specified.

Transaction Lines tab definition

Title

Description

KEMID

Required for both the From and To sides of the transaction. The KEMID to which this transaction applies.

The KEMID you specify on each line must be valid and must not be closed. You may specify only one KEMID on the From line. The From KEMID must hold sufficient units of the security to complete the transfer.

Enter the KEMID to which this security transfer applies or use the lookup  to find it. The system then displays the description of the KEMID you specified.

Etran Code

 

Required. Identifies the type of processing required for this transaction.

Enter a valid code or use the lookup  to find it. The Etran code must be valid and must have an appropriately identified General Ledger object code record—one that matches the chart for the KEMID’s General Ledger account.

Line Description (From line)

Description (To line)

Describes this transaction. Enter text as desired.

Income or Principal

Required for both the From and To sides of the transaction. Indicates whether this transaction affects income or principal. Select the appropriate value from the list.

Transaction Amount

Display only. Indicates the dollar amount of this transaction.

After the system calculates the tax lot lines records, it displays the total of the lot holding cost in this field. This value is a positive value for both assets and liabilities.

Transaction Units

Required in both the From and To sections. Indicates the number of units affected by this transaction.

Actions

Click the button to perform an action on this line. When you are adding a new line, only the add button is displayed. After you have added a line, the refresh, bal inquiry, and delete buttons are displayed.

Total Income Amount

Display only. Indicates the amount of income in all transaction lines in one section of the tab. Income totals are maintained separately in the To and From sections of this tab.

Total Principal Amount

Display only. Indicates the amount of principal in all transaction lines in one section of the tab. Principal totals are maintained separately in the To and From sections of this tab.

Total Income Units

Display only. Indicates the total number of income units in one section of the tab. Income totals are maintained separately in the To and From sections of this tab.

Total Principal Units

Display only. Indicates the total number of principal units in one section of the tab. Principal totals are maintained separately in the To and From sections of this tab.

   Buttons on this tab allow you to add, import data, refresh tax lot information, delete lines and perform balance inquiries on the KEMID(s) specified. For information and instructions, see Working in the Transaction Lines Tab above.
