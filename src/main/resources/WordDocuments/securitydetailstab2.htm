Security Details Tab

Security Details tab

This tab identifies the security to which the transaction applies and provides pertinent information about it.

Security Details tab definition

Title

Description

Security ID

Required in order to associate a security with this e-doc. Otherwise, not used.

Indicates the security affected by these transactions. Any security information you enter is for reference only; it is not used in system processing. The security record referenced must be valid and active. Only one security record may be used per e-doc.

Enter the ID or use the lookup  to find it. The system then displays the description of the security you specified along with the associated class code and description, security transaction code and description, and tax lot indicator.

Class Code

Display only. Describes the class (group) to which this security belongs.

Security Transaction Code

Display only. Indicates the security transaction code assigned to this security. The system obtains this value from the class code for the security.

Tax Lot Indicator

Display only. Indicates whether the security entered is eligible for tax lot tracking. Not relevant for cash decrease transactions.
