 Indirect Cost Recovery (ICR) Type Code document

The Indirect Cost Recovery Type document includes the Edit Indirect Cost Recovery Type and the Edit Indirect Cost Recovery Exclusion by Type tabs accessed from the lookup  screen. Multiple object codes are available to be activated for exclusion from the ICR entry. The system automatically enters data into both the Old and New sections. Selected data fields are available for editing.

Edit Indirect Cost Recovery (ICR) Type Code tab definition

Title

Description

Indirect Cost Recovery Type Code

An indirect cost recovery type code identifies an optional attribute of account that automatically generates indirect cost recovery entries

Indirect Cost Recovery Type Name

Required. The descriptive label of this specific indirect cost recovery type code

Active Indicator

Indicates whether this indirect cost recovery type code is active or inactive. Remove the check mark to deactivate this code.
