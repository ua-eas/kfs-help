Report Type tab definition

Title

Description

Active Indicator

If checked, this report type is valid and may be used on the Report Definition or Report Earn Paygroup documents. If unchecked, this report type is inactive and may not be used.

Report Description

Required. A text description for this report type.

Report Type Code

Required. A code that uniquely identifies the pay frequency for a group of employees.

Examples include:

10 - 10 pay reporting
12 - 12 pay reporting
BI - Biweekly reporting
SU - Summer Reporting
