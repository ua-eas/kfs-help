Document Layout

The Asset Depreciation Convention document includes the Edit Asset Depreciation Convention tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Asset Depreciation Convention tab definition

Title

Description

Financial Object Sub Type Code

Required for 'create new.' The financial object sub type code to be used for this asset depreciation convention.

Depreciation Convention Code

Required. The depreciation convention code ('HY' - Half Year or 'FY' - Full Year) to be used for the depreciation calculation for this object sub type

Active Indicator

 

Indicates whether this asset depreciation convention code is active or inactive. Remove the check mark to deactivate this code.
