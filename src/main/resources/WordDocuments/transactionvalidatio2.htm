Transaction Validation Reports

Each run of the GL transaction validation process creates a scrubberJob report which includes the following sections:

•      Transaction Validation Statistics showing the summary record counts of the transactions and the list of errors (if any)

•      Transaction Validation Ledger (scrubber_ledger)report

•      Transaction Validation Bad Balance Type errors (scrubber_badbal)

•      Transaction Validation Errors Report (scrubber_error)
