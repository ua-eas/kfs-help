Description

VendorExcludeJob

This job compares the vendor database to the relational EPLS file downloadable from the EPLS website. Matching records are created and can be accessed and processed via the Vendor Exclusion lookup. 

The match logic is as follows:

•      Vendor Name equals EPLS Vendor Name OR

•      Vendor Alias equals EPLS Vendor Name OR

•      Address

•      Vendor City equals EPLS City AND

•      Vendor State equals ELPS State AND

•      (Vendor Line 1 Address like EPLS Line 1 Address) OR (Vendor Zip + 4 equals EPLS Zip)
