Labor Distribution > Labor Distribution Attribute Maintenance Documents > Labor Benefits Type

Document Layout

 

The Labor Benefits Type document defines the codes to group like Benefit Object Codes. The Labor Benefits Types are referenced by the Labor Object Code Benefit document and they might include FICA, retirement, fringe benefits, etc.

Benefit Code field definitions

Title

Description

Labor Benefits Description

Required. A familiar description for the labor benefit type code.

Labor Benefit Retirement Indicator

Optional. Indicates if the object code is a retirement object code. Remove the check mark to deactivate.

    Kuali Financials does not use this information, but it may be referenced by an outside Payroll or HR system.

Labor Benefit Type Code

A unique code to identify a labor benefit type.
