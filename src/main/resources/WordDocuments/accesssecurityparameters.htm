Access Security Parameters

 

Following are the parameters that need to be reviewed and updated when Access Security is turned on.

 

Access Security Parameters

Parameter

Description

ACCESS_SECURITY_DOCUMENT_

TYPES

Lists the document types that can be selected on the Security Definition document. Document hierarchy does not work in Access Security, individual document types must be selected.

ALWAYS_ALLOW_ACCOUNT_MGR_BAL_INQUIRY_ACCESS_IND

Allows account managers to always view balance inquiry results for their accounts regardless of access permissions.

ALWAYS_ALLOW_ACCOUNT_MGR_LINE_ACCESS_IND   

 

Allows account managers to always view or edit accounting lines for their accounts regardless of access permissions.

ALWAYS_ALLOW_FISCAL_OFFICER_BAL_INQUIRY_ACCESS_IND         

 

Allows fiscal officers to always view balance inquiry results for their accounts regardless of access permissions.

ALWAYS_ALLOW_FISCAL_OFFICER_LINE_ACCESS_IND 

 

Allows fiscal officers to always view or edit accounting lines for their accounts regardless of access permissions

ALWAYS_ALLOW_INITIATOR_

DOCUMENT_ACCESS_IND  

 

Allows the document initiator (or the document initiator of the related requisition if this is a PURAP doc) to always view the document and notes/attachments regardless of access permissions.

ALWAYS_ALLOW_INITIATOR_LINE_ACCESS_IND

 

Allows the document initiator to always view the accounting line regardless of access permissions.

ALWAYS_ALLOW_PRINCIPAL_

INVESTIGATOR_LINE_ACCESS_IND         

 

Allows principal investigators to always view or edit accounting lines for their accounts regardless of access permissions.

ALWAYS_ALLOW_SUPERVISOR_

BAL_INQUIRY_ACCESS_IND

 

Allows supervisors to always view balance inquiry results for their accounts regardless of access permissions.

ALWAYS_ALLOW_SUPERVISOR_

LINE_ACCESS_IND  

Allows account supervisors to always view or edit accounting lines for their accounts regardless of access permissions.

ENABLE_ACCESS_SECURITY         

Enables access security checks in the system. There is also a system property that will need to be set by your technical staff.
