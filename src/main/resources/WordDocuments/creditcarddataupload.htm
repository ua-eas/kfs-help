Credit Card Data Upload

 >   >  >   >

   For information about procedures that apply to all KFS batch uploads on the Administration menu, see Batch Upload Basics.

The Credit Card Data Upload function is available only to the members of the KFS-SYS Operations role. When you select the Credit Card Data Upload option from the Administration menu tab, the system displays the Credit Card Data Upload document. This document allows you to import CTS Card or Corporate Credit Card transactions in XML format.
