Document Layout

The Award Status document includes the Edit Award Status tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Award Status tab definition

Title

 

Description

Award Status Code

A code associated with the particular status of an award. Examples include:

A = Awarded
R = Rejected
U = Underwritten.

Award Status Description

Required. A brief description of the status of the award which identifies the stage or progression of the award. Examples of Award Status Description are 'Awarded,' 'Underwritten,' or 'Rejected.'

Active Indicator

 

Indicates whether this award status code is active or inactive. Remove the check mark to deactivate this code.
