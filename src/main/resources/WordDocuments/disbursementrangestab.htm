Disbursement Ranges Tab

This tab displays your default campus (based on the campus code associated with the Pre-Disbursement Processor role). It also displays information about the banks in PDP, the campus associated with the displayed disbursement range for each bank and the next disbursement number to be assigned to a disbursement associated with each bank.
