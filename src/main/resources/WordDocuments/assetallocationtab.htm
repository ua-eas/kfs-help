Asset Allocation Tab

Use this tab to specify how the accounting lines should be allocated to the selected assets.

 

Asset Allocation options

Option

Description

Distribute cost by amount

Allows you to specify the amount to be allocated to each account. The Allocate amount must total the accounting line amount. Additionally, the Allocate Amount cannot be zero when this option is chosen.

Distribute cost evenly

The accounting line amounts will be distributed evenly to each asset.

Distribute cost by total cost

The accounting line amount will be distributed based on the total cost of each asset.

Distribute cost by percent

Allows you to specify what percentage of the accounting line amount should be applied to each asset.
