Proposal Type

 >  >  >  >

The Proposal Type document maintains categories that can be assigned to proposals and awards to indicate if they are new or related to a previous project at the institution. A previously non-existing Type can be added to the table or information for an existing one can be updated. The Type is primarily used with external sponsoring agencies for Contracts and Grants activities; however, it can also be used for awards internal to the institution if necessary.
