Model

 >  >  >  >  >

The Security Model document is used to create a model, or collection of definitions, and assign them to individual users (principals), groups, or roles.

When the user chooses the Model option from the menu, the system displays the Security Model Lookup screen. After the user searches for and selects a model to edit or clicks the create new button, the system presents the Security Model document.
