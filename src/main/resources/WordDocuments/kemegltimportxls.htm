KEM_EGLT_Import.xls

The KEM_EGLT_import.xls template is used to import transaction lines into the Endowment To General Ledger Transfer (EGLT) e-doc.

     Be sure to select the appropriate tab on this template. The KEM_EGLT_Import.xls template contains two sections, one for From transactions (which affect KEMID records) and the other for To transactions (which affect GL accounts). To select the appropriate section of the template, click a tab at the bottom of the Excel spreadsheet.
