Routing & Identity Management Document Type Hierarchy

 >  >  >  >

This special view combines KIM data with the KFS Document Type Hierarchy. It visually displays the parent and child relationships between KFS document types and the route nodes that are associated with each. Users may also choose to view detailed configuration for each document type, including all KIM permissions and responsibilities related to that document.

Users who are authorized to edit document types, permissions and responsibilities may access these options directly via this screen.
