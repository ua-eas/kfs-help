Transaction Lines Tab

This tab contains one or more lines that specify the KEMID(s), including all relevant financial details for the transaction as it applies to each KEMID specified.

Transaction Lines tab definition

Title

Description

KEMID

Required. The KEMID to which this cash decrease applies. Enter the KEMID or use the lookup  to find it. The KEMID you specify must be valid, must not be closed, and must own sufficient shares of the security to complete the transaction. The system then displays the description of the KEMID you specified.

Etran Code

 

Required if the Transaction Sub Type is ‘Non-Cash.’ Display only if the Transaction Sub Type is ‘Cash.’ Identifies the type of processing required for this transaction.

If the Transaction Sub Type is ‘Non-Cash,’ enter the ID or use the lookup  to find it. The Etran code entered must be valid for the KEMID specified on the line. The system automatically displays the description of the Etran code you specified.

If the Transaction Sub Type is ‘Cash,’ the system displays the security transaction code in this field.

Line Description

Describes this transaction. Enter text as desired.

Income or Principal

Required. Indicates whether this transaction pertains to income or principal. Enter the appropriate value from the list.

Transaction Amount

Required if the Transaction Sub Type is ‘Cash’ in order to calculate the gain or loss on the transaction. Display only if the Transaction Sub Type is ‘Non-Cash.’ Indicates the dollar amount of this transaction.

If the Transaction Sub Type is ‘Cash,’ enter the amount of cash received.

If the Transaction Sub Type is ‘Non-Cash,’ the system calculates and displays the amount of the transaction based on the number of units you specify.

Transaction Units

Required. Indicates the number of security units affected by the transaction. The number of transaction units must be greater than zero.

If the Transaction Sub Type is ‘Non-Cash,’ enter the number of units by which the asset is to be reduced.

Actions

Click the button to perform an action on this line. When you are adding a new line, only the add button is displayed. After you have added a line, the refresh, bal inquiry, and delete buttons are displayed.

Total Income Amount

Display only. Indicates the amount of income in all transaction lines.

Total Principal Amount

Display only. Indicates the total amount of principal in all transaction lines.

Total Income Units

Display only. Indicates the total number of income units (shares) in all transaction lines.

Total Principal Units

Display only. Indicates the total number of principal units (shares) in all transaction lines.

   Buttons on this tab allow you to add, import data, refresh tax lot information, delete lines and perform balance inquiries on the KEMID(s) specified. For information and instructions, see Working in the Transaction Lines Tab.
