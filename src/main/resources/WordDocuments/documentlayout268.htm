Object Level field definitions

Title

Description

Active Indicator

Indicates whether this object level code is active or inactive. Remove the check mark to deactivate this code.

Chart Code

Required. The chart code for the associated object level

Consolidation Code

Required. The consolidation object code associated with this object level code. Existing consolidation codes may be retrieved from the lookup icon.

Level Code

Required. The code assigned to a particular reporting level of object codes.

Object Level Name

Required. A descriptive label for this specific object level.

Object Level

Required. An abbreviated name for this level to be used in reporting where space is limited.

Report Sort Code

Required. The code that determines the default sort sequence on reports.
