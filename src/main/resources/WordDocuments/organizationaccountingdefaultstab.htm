Organization Accounting Defaults tab field definitions

Title

Description

Billing Chart Code

Required. The billing Chart of Accounts code to be used for the organization. Enter the appropriate code or use the lookup to find it.

Billing Organization Code

Required. The billing organization code. Existing organization codes may be retrieved from the lookup.

Fiscal Year

 

Required. The fiscal year of the organization accounting default. Existing fiscal years may be retrieved from the lookup.
