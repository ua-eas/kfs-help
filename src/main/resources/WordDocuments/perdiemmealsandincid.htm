Per Diem Meals and Incidentals Breakdown

 

The Per Diem Meals and Incidentals Breakdown document specifies how to allocate total per diem to breakfast, lunch and dinner when the CONUS per diem tables are imported. It is used in conjunction with parameter CONUS_MEAL_BREAKDOWN. If a total per diem match is found in this table, then it breaks out the meals as indicated, if a match is not found, then it uses the parameter to determine the breakdown.
