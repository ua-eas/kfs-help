Lookup Criteria

Milestone Report screen definition

Field

Description

Proposal Number

The proposal number associated with the milestone.

Award Chart Code

The chart code for the account associated with the award that the milestone belongs to.

Award Account Number

The ID number for the account associated with the award that the milestone belongs to..

Estimated Completion Date From

To search for milestones with an estimated completion date during a particular period of time, enter the first date in the desired range of dates. You may also use the calendar icon  to select the date.

Estimated Completion Date To

To search for milestones with an estimated completion date during a particular period of time, enter the last date in the desired range of dates. You may also use the calendar icon  to select the date.

Billed

Select whether you wish to view billed, unbilled or all milestones.

Active

Select whether you wish to view active, inactive or all milestones.
