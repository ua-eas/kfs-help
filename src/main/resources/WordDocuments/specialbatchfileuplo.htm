Special Batch File Upload

The Special Batch File Upload is used to load very large files into the General Ledger or Labor Ledger origin entry directories. Once the file has been uploaded the General Ledger Correction Process can be used to load the file for processing via the Database option. Use the same file format that is used to upload files via the General Ledger Correction Process or the Labor Ledger Correction Process.

          This functionality is not currently functioning.
