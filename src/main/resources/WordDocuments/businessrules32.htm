Business Rules

•      All fields must be filled in.

•      Chart and Organization must be valid in the Chart Organization table (ca_org_t).

•      The Reports to Chart and Reports to Organization must be valid in the Budget Construction Organization Reports table (ld_bcn_org_rpts_t).

•      The Reports to Chart and Reports to Organization must reference a valid path to the root of the hierarchy tree with no infinite loops (cycles), except for the root.

•      The Responsibility Center Code must be valid in the Chart Responsibility Center table (ca_rc_t).
