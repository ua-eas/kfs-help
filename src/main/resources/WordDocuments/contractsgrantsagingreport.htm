Contracts & Grants Aging Report

 

The Contracts & Grants Aging Report allows Collectors to return a report or print file of outstanding Contracts & Grants Invoices for customers, along with how long the invoices have been outstanding.

 

          Displays when Contracts & Grants Billing is turned on.

          Only users with the AR Collector role can access the Contracts & Grants Aging Report.
