Accounts Receivable > Customer and Attribute Maintenance Documents > Customer > Document Layout
Collections Tab

       This tab will only display if Contracts & Grants Billing is turned on.

 

Collections tab definition

Title

Description

Bankruptcy Type

Optional. The type of bankruptcy the customer has filed. Select a value from the list.

Bankruptcy Date

Optional. The date the customer filed bankruptcy. Enter a date in the format mm/dd/yyyy or use the calendar  to find it.

Bankruptcy Review Date

Optional. The date the bankruptcy review was completed. Enter a date in the format mm/dd/yyyy or use the calendar  to find it.

Stop Work

 

Optional. Check this box to indicate if work has stopped on this customer.

    Invoices will suspend when the Stop Work flag is checked.

Stop Work Reason

Optional. Enter the reason work has stopped on this customer.

    This field is required when Stop Work is checked.
