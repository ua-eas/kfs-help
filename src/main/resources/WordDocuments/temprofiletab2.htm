TEM Profile Tab

The TEM Profile tab contains information about the traveler, default accounting and his or her address. 

     This tab is display only. The information defaults from the initiators TEM Profile.

TEM Profile tab definitions

Title

Description

Traveler Type Code

Display only. The initiator’s university affiliation (employee or non-employee).

     Non-employees cannot initiate this document.

Last Update

Display only. The date on which the initiator’s TEM Profile was last updated.

Updated By

Display only. The principal name the last person who updated the initiator’s TEM Profile.

First  Name

Display only. The initiator’s first name.

Middle Name

Display only. The initiator’s middle name.

Last Name

Display only. The initiator’s last name.

Employee ID

Display only. The initiator’s employee ID.

Primary Department Code

Display only. The chart code and org code of the initiator’s primary department.

Date of Birth

Display only. The initiator’s birth date. The system masks this entry for display to members of all roles except the roles of Arranger, Profile Administrator, and Travel Manager.

Chart Code

The initiator’s default chart code.

Account Number

The initiator’s default account number.

Sub-Account Number

The initiator’s default sub-account number.

Project Code

 

The initiator’s default project code.

Street Address Line 1

Display only. The first line of the initiator’s street address.

Street Address Line 2

Display only. The second line of the initiator’s street address.

City Name

Display only. The initiator’s city.

State

 

Display only. The initiator’s state.

Zip Code

Display only. The initiator’s zip.

Country Code

Display only. The initiator’s country code.

Phone Number

Display only. The initiator’s telephone number.

Email

The initiator’s email address.
