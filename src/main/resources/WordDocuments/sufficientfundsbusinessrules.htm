Sufficient Funds Business Rules

The Account Sufficient Funds Code value must be a valid code indicating the object of the sufficient funds checking.

•      'O' Object Code

•      'L' Object Level

•      'C' Object Consolidation

•      'H' Cash at Account

•      'A' Account

•      'N' No Checking
