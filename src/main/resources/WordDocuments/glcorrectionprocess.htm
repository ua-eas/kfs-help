G/L Correction Process

If an error is encountered during any part of the validation process, the associated document(s) are not posted. Instead, error records are placed in the ScrbErr file (for more on G/L files see Accounting Cycle Files. This file can be retrieved using a G/L Correction Process (GLCP) document. This process allows for the manual correction of errors so the associated documents may be re-posted in a future batch cycle.

For information on the G/L Correction Process (GLCP) document, see G/L Correction Process (GLCP).
