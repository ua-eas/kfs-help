Importing Traveler Information

If a group includes a number of travelers, you may create a .csv file containing the travelers’ information and click  to import it into the fields on this tab.

   To access a template showing the required layout of the.csv file, see Group_Import.xlxs.

   For information about using the template to import traveler information, see Data Import Templates.
