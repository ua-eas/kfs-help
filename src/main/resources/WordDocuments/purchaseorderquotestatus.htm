Purchase Order Quote Status

 >  >  >  >

The Purchase Order Quote Status document defines the status options that may be assigned by a purchasing agent to each vendor in an existing quotation. For example, a vendor who did not submit a bid would be given the status 'No Response. For a vendor who submitted a bid that was unacceptable, the vendor would be given the status, 'Received, Bid Unacceptable.
