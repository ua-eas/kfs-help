Document Layout

The Security Reporting Group Maintenance document includes the Edit Reporting Group tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Reporting Group tab definition

Title

Description

Reporting Group

Required when initiating a document; otherwise, display only. Identifies a group of securities for reporting purposes.

Reporting Group Description

Required. The description of the reporting group code to be used in reporting. Examples include ‘Stocks,’ ‘Bonds,’ ‘Real Estate,’ etc.

Order Number

The order in which this reporting group is to appear on reports and on balance screens. No two groups may have the same order number.

Active Indicator

 

Indicates whether this reporting group is active or inactive. Remove the check mark to deactivate the group.
