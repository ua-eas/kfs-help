Principal

 >  >  >  >   >

The Security Principal document is used to assign one or more models to a specific principal and define any security exception rules that apply to this principal.

When the user chooses the Principal option from the menu, the system displays the Security Principal Lookup screen. After the user searches for and selects a principal to edit or clicks the create new button, the system presents the Security Principal document.
