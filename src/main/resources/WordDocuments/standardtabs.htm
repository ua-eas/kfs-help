Standard Tabs

While the tabs contained in various e-docs may vary from one document type to another, a set of standard tabs is included in every e-doc. All documents contain Document Overview, Notes and Attachments, Ad Hoc Recipients, and Route Log tabs. Additionally, financial transactions e-docs contain the Accounting Lines tab and the General Ledger Pending Entries tab. Another tab, Capital Edit, is included on many but not all financial transaction documents.
