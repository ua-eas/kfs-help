Effort Detail Tab

 

The Effort Detail tab is intended for use by fiscal officers and Contracts and Grants staff members. It contains all the information found in the Effort Summary tab and includes additional fields.

 

The organization of the Effort Detail tab is identical to that of the Effort Summary tab. Fields that appear only on the Effort Detail tab are explained in the table below. For all other fields, consult the table explaining the Effort Summary tab.
