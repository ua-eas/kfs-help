Post-Processing

After the Travel Manager approves the CTS Card Application, the Administrator tab on the initiator’s TEM Profile is updated:

•      The Credit Card or Agency Code field is updated with the value specified in parameter CORPORATE_CARD_CODE,

•      The Account Number field is updated with the initiator’s Employee ID,

•      The Effective Date field is updated with the date the CTS Card Application was approved and

•      The Note field is updated with “Card application approved on <date approved>, Document Number: <CTS card application document number>.”
