Reason Summary

Salary Object List with Percent Change Thresholds, Reason Code Selection

Displays summary totals for funding-related information associated with the organization selection. The summary totals are also provided on the BC Organization Reason Summary Report on the last page of each selected organization. This report allows the user to produce summary totals without having to print supporting detail.
