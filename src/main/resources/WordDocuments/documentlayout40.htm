Letter of Credit Fund Group field definitions

Title

Description

Active Indicator

Indicates whether this letter of credit fund group code is active or inactive. Remove the check mark to deactivate this code.

Letter Of Credit Fund Group Code

Required. A code to represent an abbreviation of the letter of credit (LOC) fund group description. Codes exist for most large federal departments and agencies as these agencies use LOC funding for their grants and cooperative agreements. This code is used to aggregate expenditures on all awards from a particular LOC funding group in order to determine how much cash needs to be requested from the agency. Up to four characters.

Letter Of Credit Fund Group Description

Required. A text description of the LOC fund group in order to assist in determining which LOC fund group code to assign to a given award.
