KEM_EHA_Import.xls

The KEM_EHA_import.xls template is used to import transaction lines into the Holding Adjustment (EHA) e-doc.

The basic format of the template is shown below.

KEM_EHA_Import template format

Column

Field Name

A

KEMID*

B

Etran Code*

C

Line Description

D

Income or Principal*

E

Zero-fill*

F

Transaction Amount (or blank)*

G

Unit Adjustment Amount (or zero)*
