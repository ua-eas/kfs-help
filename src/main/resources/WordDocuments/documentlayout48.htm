Document Layout

 

The Asset document includes the Asset Detail Information, Asset Location, Organization Information, Payments, Payments Lookup, Land Information, Asset Depreciation Information, Warranty, Repair History, Components, and Lookup Related Capital Asset Document(s) and View Purchasing/Financial Asset Documents tabs as well as the standard Notes and Attachments, Ad Hoc Recipients, and Route Log tabs.

 

          For more information about the standard tabs, see Standard Tabs.

          If more than ten payments exist for the asset, you may view all payments via the Asset Payment Lookup. For more information about the Lookup, see Asset Payment Lookup.
