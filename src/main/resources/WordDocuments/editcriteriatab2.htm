Edit Criteria Tab

 

The Edit Criteria tab is where users identify the rows of data they want to make changes to and specify what those changes should be. To make corrections users may create one or more groups of search criteria and corresponding modification criteria. Each group may have multiple search and modification criteria.

 

The Edit Criteria tab includes the Search Criteria section and the Modification section.
