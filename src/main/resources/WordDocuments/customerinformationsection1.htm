Customer Information Section

Customer Information section definition

Title

 

Description

Customer Number

Required. A unique number assigned to identify each vendor/organization as a customer. Either enter the customer number or use the lookup  to find it.

    This selection automatically populates field information in the Billing/Shipping tab.

Customer Purchase Order Number

If there is a purchase order associated with this invoice to this customer, enter the PO number here.

Customer Name

The name of the customer whose customer number was entered in the Customer Number field. This field is maintained by the system.

Customer Purchase Order Date

The date on the customer purchase order that indicates when it was initiated. Either enter the date or use the calendar icon  to select it.
