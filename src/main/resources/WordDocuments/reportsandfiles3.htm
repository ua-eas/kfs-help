Files dropped via an external process, such as, SFTP, or uploaded via the online Enterprise Feed Upload.

If the files are dropped into the directory by an external process, all three files must be provided.

If the Batch File Upload is used, the .done file is created by the Enterprise Feed Upload.

The .done file will be deleted by the enterpriseFeedJob after it has processed the .data and .recon files.
