Asset Status

 >  >  >  >

The Asset Status document defines codes that indicate the current status of an asset. Common values for Asset Status are:

•      A - Active and identifiable

•      C - Active and non-accessible

•      N - Non-capital active

•      O - Non-capital retired

•      R - Retired

•      S - Surplus equipment

•      U - Under construction
