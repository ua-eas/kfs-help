Correction Process Tab

The Correction Process tab is used to select the system (Database or File Upload) and the edit method (Using Criteria or Manual Edit) to be used for corrections.

After making a selection, click . This action opens available options on the other tabs of the document. The possible combinations and document tabs available to those combinations are discussed in Document Layout (LLCP).
